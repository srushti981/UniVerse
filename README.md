# UniVerse 🌍

UniVerse is a full-stack campus community platform where students can connect, discover events, access study resources, and collaborate with their peers.

## ✨ Features

- User Authentication
- Community Feed
- Campus Events
- Study Resources
- Responsive UI
- Modern React + TypeScript frontend
- REST API backend with MongoDB

## 🛠️ Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB Atlas

### Deployment
- Frontend: Vercel
- Backend: Render

## 📂 Repository

Clone the repository:

```bash
git clone https://github.com/srushti981/UniVerse.git
```

Install dependencies:

```bash
npm install
```

Run the frontend:

```bash
npm run dev
```

Run the backend:

```bash
cd backend
npm install
npm start
```
